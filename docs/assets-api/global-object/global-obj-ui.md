# $ui
* **next-app-boilerplate**를 이용하여 애플리케이션 개발 시 사용할 수 있는 **$ui** 전역 객체입니다.
* 전역 공통 객체이기 때문에 따로 **import** 하지 않고 바로 사용합니다.
* **$ui** 객체는 업무 화면에 렌더링 해야할 **UI** 요소를 **JavaScript 코드**로 쉽게 구현할 수 있게 하기위한 객체입니다. ex) Alert, Confirm, Dialog 등.







## $ui 목록
---
| Hook Name                                        | 설명                        |
| :----------------------------------------------- | :------------------------- |
| **[alert](./$ui/ui-alert.md)**    | 업무 화면에서 중요한 정보를 Dialog형태의 Alert창을 띄워주는 함수입니다.    |
| **[confirm](./$ui/ui-confirm.md)**    | 업무 화면에서 중요한 정보를 Dialog형태의 Confirm창을 띄워주는 함수입니다.   |
| **[dialog](./$ui/ui-dialog.md)**    | 업무 화면에서 Dialog형태의 새로운 창을 띄워서 좀 더 복잡한 컨텐츠를 보여주는 함수입니다.    |
