# Dialog

`mfe-lib-shared`에서 제공하는 대화상자(모달) 컴포넌트입니다.  
**현재 화면 위에 겹쳐 뜨는 패널**로, 확인·취소, 중요 알림, 짧은 폼 입력 등 사용자의 주의를 한곳에 모을 때 사용합니다. 배경은 보통 어둡게 처리되며, 닫기 전까지 뒤쪽 콘텐츠와 상호작용이 제한됩니다.
