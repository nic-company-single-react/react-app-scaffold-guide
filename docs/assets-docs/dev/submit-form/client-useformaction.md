# Form 전송 (Client Component + useFormAction)

**Client Component**에서 FormData 제출을 위한 방법으로 **useFormAction** 커스텀 훅을 사용한 방식입니다.  

**useFormAction** 훅 함수는 **react-app-scaffold** 프로젝트에서 자체적으로 제공하는 **커스텀 훅** 입니다. **폼 제출 처리 로직을 간결하게 구현**할 수 있습니다.
