# UI Component 사용하기

* **react-app-scaffold**는 **shadcn/ui** 기반의 UI 컴포넌트를 자체적으로 제공합니다.
* 모든 UI 컴포넌트는 **@components/ui** 폴더에 위치합니다. 따라서 사용 시 아래와 같이 사용합니다.
```tsx
import { Button } from '@components/ui';
```

:::tip <span class="admonition-title">UI 컴포넌트</span>목록
* UI 컴포넌트 전체 가이드 목록은 [Components](../../assets-components/index.md) 페이지를 참고하세요.
:::
